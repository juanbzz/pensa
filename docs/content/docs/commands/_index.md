pensa has 16 commands. Each does one thing.

| Command | Description |
|---------|-------------|
| [`new`](new) | Create a new Python project |
| [`add`](add) | Add a dependency |
| [`remove`](remove) | Remove a dependency |
| [`lock`](lock) | Resolve dependencies and write lock file |
| [`update`](update) | Update dependencies to latest compatible versions |
| [`install`](install) | Install dependencies from lock file |
| [`sync`](sync) | Exact venv sync (install missing + remove extras) |
| [`run`](run) | Run a command in the virtual environment |
| [`check`](check) | Validate pyproject.toml and lock file consistency |
| [`env`](env) | Show virtual environment path |
| [`list`](list) | List installed packages |
| [`show`](show) | Show details about a package |
| [`tree`](tree) | Show dependency tree |
| [`build`](build) | Build sdist and wheel |
| [`publish`](publish) | Publish to PyPI |
| [`version`](version) | Print pensa version |
